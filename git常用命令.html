git常用命令

发表于 2018-11-14 | 分类于 Git

记录我常用的git命令

配置全局账号信息

初始化文件夹为本地git仓库

格式：git init

将修改添加到暂存区

格式：git add file

可以使用git add -A一次性添加所有修改。

将暂存区内容提交到当前分支

格式：git commit -m "commit message"， -m message

克隆服务器项目到本地

格式：git clone repository-url local-path

查看代码变化

格式：git diff
